# Vegetarian Recipe Trends Analysis

## Project Overview
This project explores trends in vegetarian recipes using data analysis techniques to understand ingredient popularity, recipe patterns, and nutritional tendencies. It focuses on identifying common themes in vegetarian cooking and how recipes vary based on ingredients and preparation styles.

## Objectives
- Analyze vegetarian recipe dataset  
- Identify popular ingredients and recipe trends  
- Explore patterns in cooking styles and categories  
- Visualize ingredient frequency and relationships  
- Summarize findings in a structured research report  

## Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

## Project Structure
- `Vegetarian Recipe Trends Code .ipynb` → Main analysis notebook  
- `Vegetarian Recipe Trends Paper.pdf` → Written report and findings  
- `README.md` → Project overview and documentation  

## Key Steps
1. Loaded vegetarian recipe dataset into Python  
2. Cleaned and prepared data for analysis  
3. Identified most frequently used ingredients  
4. Analyzed recipe categories and patterns  
5. Visualized ingredient trends using charts  
6. Summarized insights in a research paper format  

## Results & Insights
- Certain ingredients (e.g., vegetables, legumes, grains) appear most frequently  
- Recipes tend to cluster into a few dominant categories (salads, soups, baked dishes)  
- Ingredient combinations show strong repetition across recipes  
- Health-oriented ingredients are commonly featured  
- Trends highlight consistent patterns in vegetarian meal preparation  

## Limitations
- Dataset may not represent global vegetarian cuisine diversity  
- Nutritional data is limited or not fully detailed  
- Recipe sources may be biased toward certain platforms or regions  
- Analysis is descriptive and does not include predictive modeling  

## Key Takeaways
This project demonstrates how data analysis can be used to uncover trends in vegetarian cooking. It highlights common ingredient patterns and provides insight into how vegetarian recipes are structured and consumed.
