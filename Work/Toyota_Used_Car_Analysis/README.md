# Toyota Used Car Analysis

## Project Overview
This project analyzes Toyota used car data to understand the key factors influencing vehicle pricing and resale value. It uses exploratory data analysis and visualization techniques to identify patterns in mileage, age, fuel type, transmission, and model type.

## Objectives
- Load and clean Toyota used car dataset  
- Perform exploratory data analysis (EDA)  
- Identify key factors affecting vehicle pricing  
- Visualize relationships between features and price  
- Summarize findings in a research-style report  

## Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

## Project Structure
- `Toyota Used Car Analysis.ipynb` → Main analysis notebook  
- `toyota.csv` → Dataset used for analysis  
- `Toyota Used Car Analysis Paper.pdf` → Written report and findings  
- `README.md` → Project overview and documentation  

## Key Steps
1. Loaded Toyota used car dataset into Python  
2. Performed data cleaning and preprocessing  
3. Conducted exploratory data analysis (EDA)  
4. Visualized distributions of price, mileage, and age  
5. Analyzed relationships between features and resale value  
6. Evaluated impact of categorical variables (fuel type, transmission, model)  

## Results & Insights
- Mileage has a strong negative relationship with resale value  
- Newer vehicles tend to retain higher prices  
- Certain models and fuel types show higher average resale value  
- Transmission type influences pricing patterns  
- Multiple variables collectively impact vehicle valuation  

## Limitations
- Dataset may not represent all Toyota models or market conditions  
- External factors such as location and demand are not included  
- Analysis is descriptive and does not include predictive modeling  
- Some categorical groups may have uneven sample sizes  

## Key Takeaways
This project demonstrates how exploratory data analysis can be used to identify the key drivers of used car pricing. It highlights how vehicle characteristics such as age, mileage, and model type significantly influence resale value.
