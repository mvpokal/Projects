# California Housing Prices Data Analysis

## Project Overview
This project explores California housing data to analyze factors that influence house prices. Using exploratory data analysis (EDA), the project examines relationships between housing features such as income, location, population, and median house value.

## Objectives
- Load and explore California housing dataset from Kaggle  
- Perform exploratory data analysis (EDA)  
- Analyze relationships between key housing features  
- Visualize trends in housing prices  
- Identify correlations between income, location, and housing value  

## Tools & Technologies
- Python  
- Pandas  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

## Project Structure
- `California Housing Prices Data Project` (Python script / notebook) → Main analysis code  

## Key Steps
1. Loaded dataset from a compressed file  
2. Inspected dataset structure using `.head()`, `.info()`, and `.describe()`  
3. Visualized distribution of median house values using histograms  
4. Analyzed relationship between distance to coast and house value using boxplots  
5. Explored income vs. house value using scatter plots  
6. Generated a correlation heatmap of key numerical features  

## Results & Insights
- Median income shows a strong positive relationship with house values  
- Homes closer to the coast tend to have higher prices  
- Certain features (income, location) are strong predictors of housing value  
- Correlation heatmap highlights key relationships between variables  

## Limitations
- Dataset may not include all external economic or regional factors  
- Analysis is exploratory and does not build a predictive model  
- Sampling was used in some plots to reduce overplotting  

## Key Takeaways
This project demonstrates how exploratory data analysis can reveal important patterns in real estate data. It highlights how socioeconomic and geographic factors influence housing prices in California.
